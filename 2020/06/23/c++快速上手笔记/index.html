c++ 快速上手笔记

c++是什么样的语言？

  静态、编译型语言。
  支持面向过程、面向对象、泛型编程。

  所谓“静态”的概念：（1）静态的编程语言在编译时进行类型检查；（2）c++ 变量的声明，事先需要指定数据类型；（3）c++ 具有static关键字，即可以修饰变量也可以修饰类的成员。

static的作用？（面试经常问）

  1. static修饰局部变量，在函数调用之间保持局部变量的值。
  2. static修饰类成员，导致只有一个该成员的副本被所有对象共享访问。
  3. static修饰的全局变量具有内部链接性，所在文件之外的文件不可访问。
  4. static修饰类的成员称为静态成员(包括数据及方法)，静态成员函数没有this指针，只能访问静态成员，以及类外部其他函数；类的普通函数具有this指针，可以访问类中任意成员。

c++ 面向对象三大特点

  c++ 是面向对象的一门语言，涵盖c语言。所以c++ 既支持面向过程也支持面向对象。
  面向对象具有三个特点：封装、继承、多态。
  （1）封装：封装的主要目的是为了数据抽象。如在一个类中，同时声明了数据成员和成员方法，使用private使得数据成员对用户不可见，通过成员方法作为接口访问数据成员并做一系列的操作。
  （2）继承：c++ 支持多继承，派生类继承基类时需要加上权限修饰符，表明该继承是public/private/protected，默认是private私有继承。多继承时不同的基类使用,隔开。
  （3）多态：同一种方法，不同对象调用执行，呈现不同的结果。派生类通过继承基类，实现基类中的virtual虚函数，使用不同派生类对象调用同一函数，结果呈现不同。

继承

（1）virtual关键字：virtual关键字可以修饰虚函数、纯虚函数；在一个类中，virtual修饰纯虚函数时，该类及未实现其所有虚函数的派生类不可被实例化；当virtual修饰虚函数时，该类及其派生类可以实例化。
（2）继承权限修饰符：private/public/protected，派生类继承基类时，需要指定继承权限，不同的继承权限可能会修改被继承之后的成员的权限；
（3）派生类可以访问基类的所有非私有成员。
（4）类中private修饰的成员只有本类以及其友元可以访问；
（5）派生类不能继承基类的构造/析构/拷贝构造函数；不能继承基类的重载运算符；不能继承基类的友元函数。

多态

  当多个派生类继承于一个基类时，同时实现该基类中相同的（纯）虚函数，不同类的不同实现方式，即是多态。

友元

  类的友元包括：友元方法、友元类。
  友元使用friend关键字修饰。
  （1）友元函数在类的内部声明，可以通过对象、类的static成员访问类的所有成员；
  （2）友元类在类的内部声明，被声明的友元类可以访问其任意成员。

内联函数

  内联函数使用inline关键字修饰。内联函数相比于普通函数，内联函数的代码在编译时被替换到调用的地方；普通函数是在运行时被调用执行。内联函数一般是代码块比较简短的函数，类中的函数都是内联函数；内联函数的主要目的是以空间换时间，能够在一定程度上提高代码的运行效率。

构造函数/析构函数/拷贝构造函数

  （1）构造函数和析构函数：构造函数分为有参和无参两种，其函数名与类名相同，无返回值类型，其作用是用来初始化对象；析构函数的函数名也与类名相同，不过需要在其前加上 ~，其作用是释放资源。
  （2）拷贝构造函数：顾名思义，拷贝、构造。其形式也是classname (const classname &obj){}；当以一个存在的对象创建一个新的对象的时候，使用拷贝构造函数；当一个类中存在动态分配内存空间时，需要使用拷贝构造函数，如果没有拷贝构造函数，编译器会自动定义一个。

public、private、protected

  （1）权限修饰符，在类中限定类成员的访问权限。
  （2）派生类继承基类时，指定以何种权限继承。
  （3）派生类以不同的权限继承基类，可能会改变其继承的成员的权限。（派生类不能够访问基类的私有成员）

const关键字

  const关键字修饰的对象（此对象不指类的实例化）不能够被修改。
  const修饰函数时，该函数不可以改变其外部声明的变量，也不能调用任何非const函数。

typedef

  该关键字用于给变量其别名。如typedef int haha;

extern

  变量/函数声明，使用extern声明的变量或者函数，才可以在任何地方使用。（在一个文件中引入另一个文件）

定义常量的两种方式

  使用const关键字和#define宏定义。

c++重载

  c++的重载包括两部分：函数重载、运算符重载；由参数个数和类型的不同，在调用时选择最佳的调用方式。

抽象类

  至少含有一个虚函数的类是抽象类。含有虚函数的类可以被实例化，含有纯虚函数的抽象类不能被实例化。

c++文件读写

  从文件读取信息（文件输入流ifstream），把信息读入文件（文件输出流ofstream）；
  与文件相关的三种类型：ifstrean,fstream,ofstream
  open()，close()

c++异常处理

  try…catch用来捕获异常并且处理，throw抛出异常。
  若使一个catch能够捕获所有的异常，则在catch(…)中使用...省略号。

模板

  模板是泛型编程的基础，以一种独立于任何特定类型的方式编写代码。
  模板包括：函数模板、类模板。
