python扩展库的使用方法

扩展库的网址：Extension Packages url
安装的时候注意库的版本、python环境的版本以及主机是32位或者64位

安装Scrapy时安装Twisted

1、豆瓣源下载：
pip install -i https://pypi.douban.com/simple scrapy
-i 豆瓣源url指定以豆瓣源安装。
本来是一次愉快并且简单的安装过程，执行上面的命令即可。
但是，可能会报错，需要我们安装twisted这个库。

于是：我需要安装Twisted。

安装Twisted

1、我们去python扩展库的网址找到twisted这个库，并且根据版本号、python的版本号、电脑主机的位数，下载相应的.whl文件。
2、在命令行执行：
pip install [whl的文件路径]
